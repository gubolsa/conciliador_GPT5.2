# Conciliação Fatura x Mobills (cards)

Interface em cards (fatura à esquerda, Mobills à direita) com painel central de sugestões.
- Concilia apenas quando valor bate exatamente (centavos)
- Sugestões de ajuste de centavos aparecem mas não conciliam
- Ignora pagamento da fatura anterior por palavra-chave/valor
